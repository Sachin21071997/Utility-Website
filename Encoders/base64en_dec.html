Base64 encoding

Base64 encoding schemes are commonly used when there is a need to encode binary data, especially when that data needs to be stored and transferred over media that are designed to deal with text.
This encoding helps to ensure that the data remains intact without modification during transport.

Base64 encoding is used commonly in a number of applications including email via MIME, as well as storing complex data in XML or JSON.
